Detailed Description

Forever running sequence that receives TLM packets from monitor and sends to responder.

Forever running sequence that setups up responder *ready toggle patterns, then receives TLM packet from monitor and sends to responder.

task axi_responder_seq::body ( )

Does all the work.

  1. In a forever loop, waits for TLM packets from monitor sends them on to slave driver / responder
  2. Creates constrained random awready toggle pattern
  3. Sends it
  4. Creates constrained random wready toggle pattern
  5. Sends it
  6. Creates constrained random arready toggle pattern
  7. Sends it
  8. In a forever loop, waits for TLM packets from monitor sends them on to responder
