Detailed Description

monitors all 5 channels for activity

  • Sends Write Address, Write Response and Read Address out on each clock
  • Sends entire Write Data and Read Data out together
  • Driver
  • * Updates agent memory on each write data beat
  • Responder
  • * Sends a read data sequence to the responder when a read address transaction is detected Main cover properties and assertions are here.
