task axi_seq::body ( )

Does all the work.

  1. Creates constrained random AXI write packet
  2. Sends it
  3. Backdoor read of memory to verify correctly written
  4. Creates constrained random AXI read packet with same len and address as write packet
  5. Sends it
  6. Verifies read back data with written data.

Set Address bus width.

AXI supports multiple bus widths, parameterized at runtime. We have to tell the sequence so we can randomize accordingly. IE: If the addr bus width is 32, don't try to use 64 bits.
